# Transcripts (Raw)

This folder contains raw transcript exports used as source material for reports.

## Layout

- Subfolders are named `YYYY-MM` (e.g. `2025-09`).
- Each subfolder contains one or more transcript `.txt` files exported from Otter AI (or a similar transcription tool).

## File naming

Transcript files follow the existing pattern:

`GMT<YYYYMMDD>-<HHMMSS>_Recording_otter_ai.txt`

Keep filenames as-is to preserve traceability back to the original recording.

## Usage

- Treat these files as immutable source-of-truth.
- If you need cleaned/normalized text, create it in a separate place (do not overwrite these raw exports).
- When referencing a transcript in a report, cite the exact subfolder and filename.
